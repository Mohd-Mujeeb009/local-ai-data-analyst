# 🧠 Local AI Data Analyst

A **ChatGPT-style, local-first AI chatbot** that helps you analyze data and images conversationally using **LLaMA 3.2**.  
All processing happens **locally on your machine** — no cloud, no data leaks.

---

## ✨ Features

- 💬 ChatGPT-style conversational interface
- 📊 Analyze CSV & Excel files with natural language
- 📈 Automatic charts and tables
- 🖼️ Image understanding (charts, screenshots, photos)
- 🧠 Conversation memory (seamless follow-ups)
- 🔒 100% local & private (runs on Ollama)
- ⚡ Fast, lightweight, and production-friendly

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **LLM Runtime:** Ollama  
- **Model:** LLaMA 3.2 / LLaMA 3.2 Vision  
- **Language:** Python  
- **Data:** Pandas, Matplotlib
